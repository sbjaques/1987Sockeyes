## How to browse

Open this folder in File Explorer or VS Code. Every `.png` and `.jpg` is viewable inline. Every `.md` file is plain text readable in any editor. The bio and article filenames encode date + publication + page + newspapers.com image ID, so you can cross-reference to newspapers.com by navigating to `https://www.newspapers.com/image/<ID>/`.
